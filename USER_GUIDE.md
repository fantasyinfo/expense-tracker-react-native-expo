## üìÖ First Day Setup

### Step 1: Set Your Initial Balances

**Why it's important:** Setting accurate initial balances ensures your app shows the correct amount of money you have.

**How to do it:**
1. Open the app
2. Tap the **Settings** icon (‚öôÔ∏è) at the bottom
3. Scroll to **Balance Management** section
4. Tap **"Set Bank / UPI Balance"**
5. Enter your current UPI/Bank balance (e.g., ‚Çπ50,000)
6. Tap **"Set Cash Balance"**
7. Enter your current cash balance (e.g., ‚Çπ5,000)
8. Tap **Save**

**üí° Pro Tip:** If you've been tracking expenses elsewhere, use **"Auto Calculate from Entries"** to automatically calculate your balances from existing entries.

### Step 2: Set Up Your Profile

**Why it's important:** Personalizing your profile makes the app feel more yours and helps with motivation.

**How to do it:**
1. Tap the **Profile** icon (üë§) at the bottom
2. Enter your name
3. Optionally add a profile picture
4. Save your changes

### Step 3: Set Your First Savings Goal

**Why it's important:** Goals give you a target to work towards and help you stay motivated.

**How to do it:**
1. Go to **Settings** ‚Üí **Savings Goals**
2. Tap **"Set Monthly Savings Goal"**
3. Enter your target (e.g., ‚Çπ10,000 per month)
4. Save

**üí° Pro Tip:** Start with a realistic goal. You can always increase it later!

### Step 4: Add Your First Entry

**Why it's important:** This gets you familiar with the entry process.

**How to do it:**
1. Tap the big **+** button in the center of the bottom navigation
2. Select **Expense** or **Income**
3. Enter the amount (e.g., ‚Çπ500)
4. Add a note (e.g., "Lunch at restaurant")
5. Select a category (e.g., "Food & Dining")
6. Choose payment method (UPI or Cash)
7. Select the date
8. Tap **Save**

**üéâ Congratulations!** You've completed your first day setup. Now let's learn how to use the app daily.

## üîß Troubleshooting

### Common Issues

#### 1. Balance Not Updating
**Problem:** Balance doesn't match actual money  
**Solution:**
- Check if you set initial balance correctly
- Verify all entries are added
- Use "Auto Calculate from Entries"

#### 2. Can't Find a Transaction
**Problem:** Can't find a specific entry  
**Solution:**
- Use search bar
- Check date range
- Try different search terms

#### 3. Export Not Working
**Problem:** Export fails or file is empty  
**Solution:**
- Check if you have entries
- Verify date range selection
- Try different export format

#### 4. Import Not Working
**Problem:** Import fails or shows errors  
**Solution:**
- Check file format (CSV or JSON)
- Verify date format (YYYY-MM-DD)
- Ensure file matches export format
- Check if file has valid data

#### 5. Categories Not Showing
**Problem:** Category icons not visible  
**Solution:**
- Refresh the app
- Check if entry has category selected
- Verify category exists
